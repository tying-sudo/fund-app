# 🚀 快速开始：启用AI智能识别功能

## ⚡ 3步上手（5分钟搞定）

### 第1️⃣步：获取智谱API Key（2分钟）

1. 打开浏览器访问：**https://open.bigmodel.cn/**
2. 注册/登录账号（支持手机号、微信）
3. 进入控制台 → 左侧菜单 **API Keys**
4. 点击 **"创建API Key"**
5. 复制生成的Key（格式类似：`xxxxxxxx.xxxxxxxx`）

> 💡 **提示**: 新注册用户通常有免费额度，足够测试使用！

---

### 第2️⃣步：配置到项目（30秒）

打开文件 `fund-app/.env.local`：

```bash
# 将下面的 your_zhipu_api_key_here 替换为你的真实API Key
VITE_ZHIPU_API_KEY=你的真实API_Key粘贴到这里
```

**示例**：
```bash
VITE_ZHIPU_API_KEY=abc123def456.ghi789jkl012
```

---

### 第3️⃣步：启动应用并使用（1分钟）

```bash
cd fund-app
npm run dev
```

然后：
1. 打开浏览器访问 **http://localhost:5173**
2. 点击底部导航栏的 **"持仓"** 标签
3. 看到右上角的 **📷 导入图标** → 点击它！
4. 在弹出的二级菜单中选择 **"🔥 AI智能识别"** ⭐
5. 选择一张支付宝/微信基金截图
6. 点击 **"🚀 AI智能识别"** 按钮
7. 等待1-2秒... 完美识别结果出现！✨

## 🔧 常见问题排查

### Q1: 提示"请先配置智谱API Key"

**原因**: `.env.local` 文件中的Key未填写或格式错误

**解决**:
1. 确认文件路径正确：`fund-app/.env.local`
2. 确认没有多余空格或引号
3. 重启开发服务器（修改.env需要重启才生效）：`Ctrl+C` 然后 `npm run dev`

### Q2: 提示"API请求失败: 401"

**原因**: API Key无效或已过期

**解决**:
1. 回到 https://open.bigmodel.cn/ 控制台检查Key状态
2. 如果显示"已禁用"，删除旧Key，重新生成一个
3. 更新 `.env.local` 并重启服务器

### Q3: AI识别速度慢？

**正常情况**: 1-2秒/张（取决于网络和图片大小）

**优化建议**:
- 使用清晰截图（避免模糊照片）
- 图片大小控制在 5MB 以内
- 避免在网络高峰期使用（如晚8-10点）

### Q4: 如何切换回本地OCR？

两种方法：

**方法1**: 二级菜单选择
- 点击导入图标 → 选择"📷 本地OCR识别"

**方法2**: 代码默认值（可选）
编辑 `Holding.vue` 第1089行：
```typescript
const ocrMode = ref<'local' | 'ai'>('local')  // 改为 'local'
```
